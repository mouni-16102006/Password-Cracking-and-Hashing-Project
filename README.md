# Password Hashing & Cracking Simulation

This project demonstrates the fundamentals of password security, cryptographic hashing, and how dictionary attacks operate.

## Features
* **Hash Generation:** A Python script utilizing the `hashlib` library to generate MD5 and SHA-256 hashes.
* **Dictionary Attack Engine:** A custom script that loops through a wordlist, hashes entries, and compares them dynamically to identify weak passwords.

## Skills Learned
* Cryptographic Hashing (MD5, SHA-256)
* Automation and String-Matching Logic
* Troubleshooting Environment & String Variables
